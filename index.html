Visualizations

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this we first pulled data from the open weather API to make a dataset of over 500 cities.

After assembling the data we used Matplotlib to plot the various aspects of the weather vs Latitude. Factors we looked at included Max Temperature, Humidity, Cloudiness, and Wind Speed. This site provides the source data and the visualizations created as a part of the analysis as well as explanations and descriptions of any trends and correlations witnessed.
